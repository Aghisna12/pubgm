Guide To Install Mod Apk

Game : PUBG Mobile

PackageName : com.vng.pubgmobile

1. BackUp Saved Data

  • Goto /sdcard/Android/data/com.vng.pubgmobile
  • Rename to/sdcard/Android/data/com.vng.pubgmobile_bak

2. BackUp OBB Data

  • Goto /sdcard/Android/obb/com.vng.pubgmobile
  • Rename to/sdcard/Android/obb/com.vng.pubgmobile_bak

3. Uninstall Original Apk

  • Goto Setting Find App PUBG Mobile and Uninstall it

4. Install Mod Apk

  • Goto Downloaded Mod Apk and Install it

WARNING! After installation finish, don't open the app!

5. Restore Saved Data

  • Goto /sdcard/Android/data/com.vng.pubgmobile_bak
  • Rename to/sdcard/Android/data/com.vng.pubgmobile

6. Restore OBB Data

  • Goto /sdcard/Android/obb/com.vng.pubgmobile_bak
  • Rename to/sdcard/Android/obb/com.vng.pubgmobile

All is Done? Ok, you can open Modded PUBG Mobile Now.

Enjoy!
